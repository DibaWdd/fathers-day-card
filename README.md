# Father's Day Card 🎉

An interactive Persian Father's Day greeting card built with HTML, CSS, and JavaScript.

## Features
- 🎨 Colorful animated design
- 🎊 Confetti celebration effect
- 📱 Responsive and mobile-friendly
- 🌐 Persian (Farsi) language support with RTL layout

## Live Demo
👉 **[Click here to see the card](https://dibawdd.github.io/fathers-day-card/fathers_day_card.html)**

## Technologies Used
- HTML5
- CSS3 (animations, gradients, flexbox)
- JavaScript (confetti effect)
- Google Fonts (Vazirmatn for Persian text)
